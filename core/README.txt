OpenAlea.Core
-------------

OpenAlea.Core provides :
   - OpenAlea Package Manager
   - DataFlow construction

It doesn't include any GUI interface (See OpenAlea.Visualea for that).

Dependances
-----------

Python >= 2.4   (See http://www.python.org)
OpenAlea.Config (See http://openalea.gforge.inria.fr)


Installation
------------

python setup.py install


Documentation
-------------

In a python interpreter : 

   import openalea.core
   help(openalea.core)
