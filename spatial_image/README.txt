===== About =====

=== Description ===

spatial_image is an extension of numpy.ndimage package

 
=== Content ===

The spatial_image package contains :

  *Meta informations files (README.TXT, AUTHORS.TXT, LICENSE.TXT).
  *Python package (src/ndimage).
	- interface definition for SpatialImage
	- functions to serialize nspatial images
	- simple objects to provide views on spatial images

===== Installation =====

=== Download ===

use svn and check out the spatial_image directory

=== Requirements ===

* Python 2.4 or higher
* numpy

=== Installation ===

python setup.py install
